# EPA Demo Website

This is a full-stack website project with a React frontend and Express.js backend.

## Project Structure

- `Frontend/` - React website (what users see)
- `Backend/` - Express.js server (handles data and logic)

## How to Run

### Run the Backend (Server)
1. Go to the Backend folder
2. Run `npm install` then `npm run dev`
3. Server runs on http://localhost:5000

### Run the Frontend (Website) 
1. Go to the Frontend folder  
2. Run `npm install` then `npm run dev`
3. Website runs on http://localhost:5173

## What This Project Does

- Shows news and events
- Has an admin panel to manage content
- Stores data in MongoDB database
- Allows image uploads

## Technology Used

- Frontend: React, TypeScript, Vite
- Backend: Express.js, MongoDB, Node.js
